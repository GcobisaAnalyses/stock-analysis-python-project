# IBM Python Project - Stock Analysis

## Overview
Analysis of Tesla (TSLA) and GameStop (GME) stock 
data using Python. Includes stock data extraction, 
revenue web scraping, data visualization, and 
comparative financial analysis.

## Technologies Used
- Python
- yfinance
- BeautifulSoup
- Pandas
- Matplotlib
- Jupyter Notebook

## Project Structure
- `notebooks/` - Main Jupyter notebook
- `screenshots/` - Results and outputs
- `requirements.txt` - Python dependencies

## Skills Demonstrated
- Web scraping with BeautifulSoup
- Financial data extraction via yfinance API
- DataFrame manipulation with Pandas
- Data visualization with Matplotlib
